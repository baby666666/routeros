# Routeros学习记录
1、打开安装RouterOS脚本网站：
https://mikrotik.ltd/

2、通过ssh连接服务器ubuntu和debian都可以，在命令窗口输入shell命令：
VERSION=7.21.2 bash <(curl https://mikrotik.ltd/chr.sh)

3、大概一分钟后，通过winbox连接上在服务器上安装好的RouterOS CHR系统，关闭除winbox的所有协议。
